# 22 — Voice-Rewrite Polish Decisions

Synthesis of four post-rewrite editor reports (`review/18` conciseness,
`review/19` clarity, `review/20` flow, `review/21` intention) into a
polish playbook.

This polish is **calibration, not structural rewrite**. Voice is
installed. Lead-with-claim is held. The eight-section template is gone.
What remains is residue.

## Convergent diagnoses

1. **Doubled closures.** Several chapters land their closing image, then
   add a thesis-restatement coda. Ch 1, 6, 8, 17 are the worst.
2. **Commentator stacks.** Some chapters tour 4–6 readings before
   landing the chosen one. Chs 2, 5, 11, 18.
3. **Modern-source stacks.** Three citations doing one job. Chs 1, 6,
   8, 13.
4. **Buried leads.** Six chapters bury the lead behind architecture-
   preview meta-commentary or scene-setting. Chs 4, 5, 9, 11, 14, 16.
5. **Sanskrit density spikes.** Sanskrit transliteration without
   English-first rendering hits the reader before the gloss lands.
   Chs 1, 5, 7, 13, 18.
6. **Architecture-preview meta-commentary.** "*The chapter will…*"
   "*Three things hold this chapter together.*" Chs 4, 5, 8, 9, 14, 17.
7. **Subsection / bolded-tag residue.** Headers and bolded mini-labels
   doing structural work the prose should be doing. Chs 5, 13, 14.

## Manuscript-wide cuts (apply wherever they appear)

- **Cut doubled closures.** If a chapter closes once on an image, do
  not add a coda. Cut.
- **Compress commentator stacks.** Engage 2–3 substantively. Cite the
  rest in passing.
- **Compress modern-source stacks.** One named researcher for one
  point. Don't pile three studies that say the same thing.
- **Cut architecture-preview meta-commentary.** "*The chapter will*"
  / "*Three things hold this chapter together*" / "*I name them in
  order*" — gone.
- **Sanskrit gloss-first.** First appearance: English rendering or
  inline gloss BEFORE the transliteration becomes load-bearing. The
  reader should know what the term means in the line where it lands.
- **Drop bolded mini-labels** that mark sub-cases. Continuous prose.
- **Drop "I will" / "I am about to" / "Here I want to" framings.**
  Begin the move, don't announce it.

## Process

The 18 polish agents run in parallel. Each:

1. Reads this document (review/22), focuses on its chapter's
   directive.
2. Reads the four post-rewrite editor reports (review/18–21) for
   chapter-specific context where useful.
3. Reads its chapter and applies the directives + manuscript-wide
   cuts.
4. Edits in place via the Edit tool (preferred over Write).
5. Reports original/final word count + what was applied.

Hard floors hold throughout.
